# HomeSOCAnalystLab
<h1>Home SOC Analyst Lab</h1>

<h2>Description</h2>
Created an entire SOC Analyst home lab complete with firewall, a router, hosts, an attack box, network monitoring, and a log aggregation tool. This up-and-running system deployed agents and generated telemetry received by the logging tool which allowed for practice with detecting and blocking simple attacks.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Powershell</b> 
- <b>LimaCharlie</b>
- <b>Sysmon</b>
- <b>Sliver</b>

<h2>Environments Used </h2>

- <b>Windows/VMWare</b>
- <b>Kali Linux/Ubuntu</b>
